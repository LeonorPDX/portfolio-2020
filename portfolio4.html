Project: Cookbook Community

MVC web app built with Sinatra

Cookbook Community is an MVC web app that was built with Sinatra and ActiveRecord. It uses sessions to keep track of users as they log in and out, and each user has full CRUD abilities with resources that belong to that user.

This project is a recipe sharing site where users can contribute recipes to a shared index of all recipes, update or delete their own recipes, and also create a collection of saved recipes in their own “Cookbook.” Users can browse recipes by type, view recipes by author, and view other users’ Cookbooks.
